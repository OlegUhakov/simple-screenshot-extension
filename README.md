# Simple Screenshot Extension

A Chrome/Edge extension (MV3) for taking screenshots with annotations.

## Features

- **Full Screen** — capture the visible area of the current tab
- **Select Area** — drag to select a region to capture
- **Timer** — delay of 0/3/5/15 seconds before capture
- **Editor** — annotate screenshots after capture:
  - Tools: pencil, brush, line, rectangle, circle, arrow, text, eraser, blur
  - Color picker and stroke width
  - Undo / Clear
  - Save as PNG / Copy to clipboard

## Installation

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the project folder

## Usage

1. Click the extension icon in the toolbar
2. Choose **Select Area** or **Full Screen**
3. Optionally set a timer
4. The editor will open after the screenshot is taken
5. Add annotations and Save or Copy the result
